Consideraciones

This folder contains all source files that you can use to build custom template.

Altair is build on LESS (CSS preprocessor), uses Bower as a package manager and uses Gulp for its build system.

LESS

For easy customization Less files are included in Altair. This files are located in /src/assets/less/ folder.

Folder structure and description:

Folder Description
components/ Custom components
md/ Material design styles (buttons, colors, cards, shadows etc.)
pages/ Page specific styles
partials/ Styles for header, sidebars and top bar
plugins/ Custom styles for plugins
altair_admin.less Main styles
kendo_ui_custom.less KendoUI components
main.less Main less file processed by gulp less_main task
print.less Print styles
uikit_custom.less UIkit custom styles
variables_mixins.less Variables like height/width/colors and some mixins used in this template

Compiled code (/dist)

This folder contains production ready files (minified JavaScript, compressed CSS, HTML files).

HTML Structure

Unidad 1: Conceptos Básicos

Introduccion

To hide sidebar (menu) on page load remove class sidebar_main_open from <body> element.
Main sidebar is hidden by default when the viewport is smaller than 1220 pixels

Ejemplos

To show mini sidebar on page load remove sidebar_main_open sidebar_main_swipe classes from <body> and add sidebar_mini class to <body>

Materiales Adjuntos

To show slim sidebar on page load remove sidebar_main_open sidebar_main_swipe classes from <body> and add sidebar_slim sidebar_slim_inactive class to <body>

Evaluación

HTML

To enable accordion mode add accordion_mode class to <aside id="sidebar_main" class="accordion_mode"></aside>

AngularJS

To enable accordion mode add $rootScope.menuAccordionMode = true; to run block in app/app.js
